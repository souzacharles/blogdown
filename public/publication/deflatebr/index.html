Pacote deflateBR

O pacote deflateBR foi criado por Meireles (2018) e obtém séries de índices de preços atualizados da API do Instituto de Pesquisa Econômica Aplicada (IPEA).

Carregando o pacote

library(deflateBR)
deflate(nominal_values = 100, nominal_dates = as.Date("2000-01-01"), 
        real_date = "01/2019")
## 
## Downloading necessary data from IPEA's API
## ...
## [1] 322.1112

Por padrão usa o API IPEA.

Para alterar o índice acresenta-se o argumento index.

deflate(nominal_values = 100, nominal_dates = as.Date("2000-01-01"),
        real_date = "01/2019", index = "igpm")
## 
## Downloading necessary data from IPEA's API
## ...
## [1] 397.2442
deflate(nominal_values = 100, nominal_dates = as.Date("2000-01-01"),
        real_date = "01/2019",  "inpc")
## 
## Downloading necessary data from IPEA's API
## ...
## [1] 329.537

Deflacionando series

dados <- data.frame(reais = seq(1, 5),
                    datas = seq.Date(from = as.Date("2001-01-01"), 
                                     by = "month", length.out = 5),stringsAsFactors = FALSE)
reais datas
1 2001-01-01
2 2001-02-01
3 2001-03-01
4 2001-04-01
5 2001-05-01 
deflate(dados$reais, dados$datas, "01/2019", "ipc")
## 
## Downloading necessary data from IPEA's API
## ...
## [1]  2.967902  5.898014  8.811958 11.683967 14.480809

Outras funções

Para facilitar o uso de cada um dos cinco índices de preços, também incluí no pacote deflateBR cinco funções auxiliares. Em vez de usar a função deflate com o argumento index = "ipca" para deflacionar pelo IPCA, é possível fazer isso:

ipca(dados$reais, dados$datas, "01/2019")
## 
## Downloading necessary data from IPEA's API
## ...
## [1]  3.039514  6.044558  9.025311 11.988215 14.898877
igpdi(dados$reais, dados$datas, "01/2019")
## 
## Downloading necessary data from IPEA's API
## ...
## [1]  3.598098  7.161123 10.705435 14.160172 17.502859

Por fim, o pacote deflateBR também contém uma função simples, chamada inflation, para identificar a variação percentual na inflação entre dois períodos. Se quisermos saber qual foi a taxa de inflação em 2017 segundo o IPCA, por exemplo, basta rodar:

inflation("01/2000", "01/2019", "ipca")
## 
## Downloading necessary data from IPEA's API
## ...
## [1] 222.1112
